# JS-Matrix
A basic matrix math library.

## Installation
```
npm install @rewforen/js-matrix
```

## Usage
```JS
const { Matrix } = require('js-matrix');

let mat = new Matrix(2, 3, [1, 2, 3, 4, 5, 6]);
```
